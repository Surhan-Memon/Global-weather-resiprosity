# Global-weather-resiprosity
This project focuses on analyzing global weather data to understand patterns in temperature, rainfall, and overall climate trends. Using Python and Pandas, I cleaned and processed the dataset by handling missing values, correcting data types, and ensuring consistency for analysis.

I performed exploratory data analysis (EDA) to uncover key insights about climate behavior across different regions. Some highlights of the analysis include:

-> Temperature trends: Calculated average temperatures over time, identified seasonal variations, and visualized changes using line charts.

-> Rainfall patterns: Analyzed total and average rainfall across different countries and months, highlighting regions with extreme rainfall events.

-> Extreme weather events: Identified and visualized occurrences of unusually high or low temperatures and rainfall to study climate anomalies.

-> Comparisons across regions: Used grouping and aggregation to compare weather patterns across continents, countries, and climate zones.

-> Visualizations: Created bar charts, histograms, and line plots for key metrics, making trends and patterns easy to interpret.

This project demonstrates data cleaning, preprocessing, aggregation, and visualization using Python tools. It provides practical experience with handling real-world datasets and generating insights that can inform studies in climate science, environmental research, or policy-making.
